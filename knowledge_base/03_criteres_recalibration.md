# Critères de recalibration d'un instrument de diagnostic

## Déclencheurs de recalibration

Une recalibration est requise dans les cas suivants :
- Violation d'une règle de Westgard de type systématique (2-2s, 4-1s, 10x).
- Dépassement d'un intervalle de temps fixe depuis la dernière calibration
  (typiquement 24h à 30 jours selon le type d'analyseur et le fabricant).
- Changement de lot de réactif ou de consommable critique.
- Après toute intervention de maintenance corrective sur le capteur.
- Résultat de contrôle qualité externe (EQA/EEQ) hors tolérance.

## Fréquence typique selon le type de dérive

- **Dérive lente et systématique** (accumulation sur plusieurs jours/semaines) :
  recalibration programmée selon le calendrier standard suffit généralement.
- **Dérive brutale** (saut soudain de valeur) : recalibration immédiate requise,
  indépendamment du calendrier programmé — probable cause matérielle.

## Validation post-recalibration

Après recalibration, un minimum de 2 à 3 contrôles qualité consécutifs dans les
tolérances est requis avant de considérer l'instrument à nouveau validé pour
la production de résultats patients.
